The Problem

Long-horizon video understanding faces a fundamental challenge: important events are rare, localized, and buried in hours of background footage. Standard approaches waste 99%+ of compute on irrelevant frames.


Our framework treats event discovery as hierarchical signal detection, using physics-inspired energy functionals and sparse optimization to find the needle in the haystack.

Energy Functional

E(W) = α1 · φmotion + α2 · φinteraction + α3 · φscene + α4 · φuncertainty

Background = Low Energy

Equilibrium states with minimal motion, interaction, and scene change. Filtered cheaply by hierarchical thresholding.

Events = High Energy

Excitations above adaptive thresholds. Selected via greedy submodular maximization with 1-1/e approximation guarantee.

Quick Start

# Install
git clone https://github.com/meshal-alawein/event-discovery-framework.git
cd event-discovery-framework
pip install -e ".[all]"

# Detect events
event-discovery detect video.mp4 --method hierarchical --top-k 10

# Compare methods
event-discovery compare video.mp4 annotations.json -o results/

Python API

from event_discovery.methods import HierarchicalEnergyMethod, EnergyConfig

config = EnergyConfig(top_k=10)
method = HierarchicalEnergyMethod(config)
events = method.process_video('video.mp4')

for event in events:
    print(f"Event: {event.start_time:.1f}s - {event.end_time:.1f}s")
